Về ScholarHub

Nền tảng chia sẻ và tìm kiếm tài liệu học tập trực tuyến dành cho sinh viên Đại học Đại Nam.

Tính năng sinh viên
  • Tìm tài liệu, lọc theo môn học / năm học / loại
  • Xem mô tả, link tải, đánh dấu yêu thích
  • Đăng tải tài liệu (chờ duyệt Pending)
  • Đếm lượt xem qua PUT API MockAPI
Công nghệ

JavaScript thuần, Fetch API, jQuery, Bootstrap 5, MockAPI.io — deploy Vercel.
